# Spring AI Learning Project

## Overview
This project is designed to learn Spring Boot and AI integration by building multiple AI-powered features:

1. **AI Photo Generator** - Generate images from text prompts.
2. **Ask AI** - Ask questions and get intelligent AI responses.
3. **Recipe Generator** - Generate recipes based on ingredients or dish types.

## Technologies Used
- Java 17+
- Spring Boot
- REST APIs
- AI API integration (OpenAI / Hugging Face / other)
- JSON handling
- Optional frontend: React.js or Thymeleaf

## Features
### 1. AI Photo Generator
- Input a text description
- Receive an AI-generated image
- Save or display image in the app

### 2. Ask AI
- Input a question
- Receive an AI-generated answer
- Useful for chat or knowledge-based queries

### 3. Recipe Generator
- Input ingredients or dish name
- Receive AI-generated recipe
- Includes preparation steps and ingredients list

## Learning Goals
- Learn Spring Boot REST API development
- Learn to integrate external AI APIs
- Handle JSON requests/responses
- Connect backend with frontend for a fullstack experience
